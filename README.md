# 知错 - 智能错题管理系统

## 项目概述
「知错」是一款聚焦学习场景的智能化错题管理系统，核心目标是帮助不同学习阶段的用户高效完成错题的记录、整理、复习闭环，通过AI技术赋能学习过程，强化知识薄弱环节，最终提升学习效率与知识掌握深度。系统围绕错题全生命周期管理设计，覆盖从错题收集到智能复习的全流程，为学习者提供一站式错题解决方案。

## 核心功能
### 1. 多维度错题管理
- 支持**手动录入**和**图片识别录入**两种核心添加方式，适配不同使用场景；
- 完整记录错题信息：包含题目内容、作答记录、正确答案、解析、错误原因标注等；
- 支持按**科目、章节、题型、错误类型、掌握状态**等多维度分类、标签化管理，快速检索定位。

### 2. 个性化智能复习
- 双复习模式：「原题复盘」（聚焦错误本身）+「举一反三」（拓展同类题型）；
- 基于艾宾浩斯遗忘曲线，智能推送复习计划，记录错误次数与掌握进度；
- 自动生成相似题型练习，强化知识点迁移应用能力。

### 3. AI辅助学习能力
- OCR图片识别：精准提取题目文本、科目、题型等关键信息，减少手动录入成本；
- 智能解题助手：针对错题提供分步解题思路、核心知识点拆解、易错点提醒；
- 知识点关联推荐：基于错题自动关联相关知识点，补充知识盲区。

### 4. 数据化学习分析
- 可视化数据面板：展示错题数量、错误趋势、各科目错题占比等核心数据；
- 薄弱项分析：自动定位高频错误知识点、易错题类型，生成学习改进建议；
- 学习进度追踪：记录复习完成率、掌握程度变化，直观呈现学习效果。

## 技术特点
- 跨端适配：响应式设计，兼容PC端、移动端（手机/平板），保证多设备使用体验一致；
- AI能力集成：融合OCR识别、自然语言处理（NLP）、个性化推荐等AI技术；
- 轻量化操作：界面简洁直观，核心功能一键触达，降低学习使用门槛；
- 数据安全：本地+云端双存储模式，支持错题数据导出/备份，保障数据不丢失。

## 适用场景
- **学生群体**：小学/初中/高中/大学生整理各科错题，针对性复习提分；
- **备考人群**：考研、考公、考证等考生系统化梳理错题，强化薄弱考点；
- **教师群体**：收集学生高频错题，分析教学重点与学生知识漏洞；
- **自主学习者**：跟踪学习轨迹，通过错题复盘提升自主学习效率。

## 核心价值
以「错题」为核心，通过智能化、系统化的管理方式，让用户从“盲目刷题”转向“精准复盘”，真正实现「知错能改、举一反三」，用更少的时间提升学习效果。
